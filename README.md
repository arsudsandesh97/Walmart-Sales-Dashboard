# 📖 Project Overview

The **Walmart Sales Dashboard** is an interactive Business Intelligence solution built using **Power BI** to analyze Walmart's sales performance and profitability across multiple years.

The dashboard helps identify business trends, monitor key performance indicators (KPIs), and uncover insights related to product categories, monthly sales, and yearly profits. It is designed to support data-driven decision making through clean visualizations and interactive reporting.

---

# 🎯 Problem Statement

Businesses generate large volumes of sales data every day, making it difficult to identify performance trends without proper visualization.

This project aims to:

- Analyze sales and profit performance
- Compare yearly business growth
- Identify top-performing product categories
- Discover monthly sales patterns
- Present business insights through an interactive dashboard

# 📊 Key Business Insights

## 📅 Yearly Profit Analysis

| Year | Profit |
|------|---------:|
| 2011 | **$148K** |
| 2012 | **$140K** |
| 2013 | **$187K** |
| **2014** | **$251K** ⭐ |

- **2014** recorded the highest annual profit.
- Total Revenue: **$725.46K**

## 📦 Category Performance

- 🏆 Highest Profitable Category:
  - **Binders** → **$16.10K Profit**

- 📱 Highest Selling Category:
  - **Phones**
  - Contributed **35%** of total sales.
